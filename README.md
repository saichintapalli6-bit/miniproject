# Real Estate Land Price Prediction System

## Features
- User & Admin Registration/Login
- GPS-based location tracking
- ML-powered price prediction
- Admin dashboard with analytics
- Search history tracking
- State-wise price comparison

## Installation

1. Install Python 3.8 or higher

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

4. Open browser and navigate to:
```
http://localhost:5000
```

## Features Overview

### User Dashboard
- Select state and city from dropdown
- Enter property details (square feet, soil type, water level)
- GPS location detection or manual input
- Real-time price prediction

### Admin Dashboard
- View all registered users
- Track search history
- View statistics and analytics
- Compare land prices across years

## Dataset Information

The application includes data for:
- 11 Indian states
- 50+ major cities
- GPS coordinates for location-based pricing
- 6 soil types
- Historical price trends

## Technology Stack
- Backend: Flask (Python)
- Database: SQLite
- Frontend: HTML, CSS, JavaScript
- ML Algorithm: Multi-factor regression model

## Security Note
Change the SECRET_KEY in app.py before deploying to production!
